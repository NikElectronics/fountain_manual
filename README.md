## 📋 О проекте

Веб-интерфейс для контроллера управления фонтаном **«МОЙ ФОНТАН»** — полноценная система автоматизации с поддержкой до **32 форсунок**, **50 программами** по **32 цикла**, музыкальным сопровождением и RGB-подсветкой. Управление и настройка осуществляются через удобный веб-интерфейс, доступный с любого устройства в локальной сети.

<div align="center">

| 🔧 Характеристики | Значение |
|:---|:---|
| Управление форсунками | до 32 шт. |
| Количество программ | 50 |
| Циклов в программе | до 32 |
| Диапазон частот | 0–250 Гц |
| Протоколы связи | Modbus RTU (RS-485), DMX, Wi-Fi |

## 🔌 Совместимое оборудование

- **Частотные преобразователи:** Innovert, Zuked (+ режим симулятора)
- **Протоколы:** Modbus RTU (RS-485), DMX (RGB-подсветка)
- **Датчики:** анемометр, уровнемер (Min/Max), RTC (часы реального времени)
- **Активная колонка** для музыкального сопровождения
- **Модули реле** для управления подсветкой и клапанами

## 🚀 Быстрый старт

1. **Подключите контроллер** к питанию 12 В
2. **Подключитесь по Wi-Fi:** точка доступа `fontan` (IP: `192.168.4.1`) или настройте подключение к вашей сети
3. **Откройте веб-интерфейс** в браузере
4. **Настройте параметры:** количество форсунок, частоты, расписание
5. **Загрузите музыку** на SD-карту через файловый менеджер
6. **Сохраните настройки** и запустите фонтан!
