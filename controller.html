Внешний вид контроллера "МОЙ ФОНТАН"

Рис. 1 - Внешний вид контроллера управления фонтаном

Основные элементы контроллера:

  1. Дисплей - отображает текущее состояние системы и меню управления
  2. Кнопки навигации - для перемещения по меню и выбора параметров
  3. Разъемы подключения - для соединения с форсунками, подсветкой и другими устройствами
  4. Индикаторы состояния - отображают текущий режим работы системы
  5. Разъем питания - для подключения к источнику питания 12В
